# McDonalds Nutrition Analysis

## Overview
This project analyzes nutrition facts of McDonalds food products using Python,
Pandas, SQLite, and SQL.

## Key Questions
1. Which product contains highest amount of protein and lowest fat, and therefore suitable for people weight training?
2. Which product contains highest amount of sodium?

## Tool Used
Python
Pandas
SQLite
SQL
Seaborn

## Summary
1. ______ is the best chioce for people needing more protein and relatively less fat if they want to eat McDonalds.
2. ______ is the item containing most sodium content, and therefore might cause water retention.
